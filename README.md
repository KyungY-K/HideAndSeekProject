## Description
  한국에서 제조하는 여러가지 술들을 종합적으로 모아서 판매하는 쇼핑몰을 만들었다.
요즘 트렌드에 맞게 구독 서비스도 포함시켜 단순히 구매하기만 할 수 있는 쇼핑몰이 아닌 정기적으로 구매를 하는 고객층을 확보하는 홈페이지를 제작하고자 했다.
